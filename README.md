# WRSR Calculator

**Optimization calculator for Workers & Resources: Soviet Republic**

A command-line tool to help you plan optimal production chains, resource allocation, and city layouts for maximum efficiency in your Soviet republic.

## Installation

### Option 1: Download Release (Recommended)
1. Download the latest release from the [Releases page](https://github.com/Tamiscola/SovietRepublicPlanner/releases/tag/v1.0.0)
2. Extract the zip file
3. Run `WRSR-Calculator.exe` (Windows) or `./WRSR-Calculator` (Linux/Mac)

### Option 2: Build from Source

**Prerequisites:** .NET 6.0 SDK or higher ([Download here](https://dotnet.microsoft.com/download))

```bash
# Clone the repository
git clone https://github.com/Tamiscola/SovietRepublicPlanner.git

# Navigate to directory
cd SovietRepublicPlanner

# Build and run
dotnet build
dotnet run
